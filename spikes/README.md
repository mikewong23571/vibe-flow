# Pre-Project Exploration Stages

`spikes/` 保存的是正式项目编码开始前的探索资产。

这里的目录名遵循统一规则：

```text
preproject_stageN_<theme>
```

其中：

* `preproject` 表示这批资产属于正式项目启动前的探索阶段
* `stageN` 表示探索推进顺序
* `<theme>` 表示该阶段的核心验证主题
